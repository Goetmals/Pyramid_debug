FEATURES  
  • Triple 10-bit D/A Converters
  • 80 (THS8135-80) or 240 MSPS (THS8135-240) Operation
  • YUV/RGB Configurable Blanking for Full (0-1023) or Video (ITU-R.BT601) –compliant Input Code Range
  • Generic triple DAC mode for non-video applications
  • Direct Drive of Doubly-terminated 75 Load into Standard Video Levels,irrespective of Operation with Full or Video Input Code Range
  • 3x10 bit 4:4:4, 2x10 bit 4:2:2 or 1x10 bit 4:2:2 (ITU-R.BT656) multiplexed YPbPr/GBR input modes
  • Bi-level (EIA) or Tri-level (SMPTE) Sync Generation
  • Integrated Insertion of Sync-On-Green/Luminance or Sync-On-All Channels
  • Internal Voltage Reference
  • See also: THS8136 (8bit, pin-compatible)
